# Excel Data Processor - Windows版本

## 🚀 快速开始

### 如果您已有exe文件
双击 `ExcelDataProcessor.exe` 即可使用。

### 如果您需要构建exe文件

#### 方法1: 一键构建（推荐）
1. 确保已安装Python 3.8+
2. 双击 `build_windows.bat`
3. 等待构建完成
4. 在 `dist` 文件夹中找到 `ExcelDataProcessor.exe`

#### 方法2: 命令行构建
```cmd
python build_exe_windows.py
```

## 📋 功能特性

- ✅ **文件导入**: 支持Excel (.xlsx, .xls) 和CSV文件
- ✅ **字段选择**: 选择需要的数据列
- ✅ **自定义字段**: 添加新的数据列
- ✅ **数据预览**: 实时预览处理结果
- ✅ **文件输出**: 生成新的Excel文件
- ✅ **现代界面**: 直观易用的图形界面

## 🖥️ 系统要求

- Windows 7/8/10/11
- 2GB 内存（推荐4GB）
- 200MB 磁盘空间

## 📖 使用方法

1. **启动程序**: 双击 `ExcelDataProcessor.exe`
2. **导入文件**: 点击"选择Excel/CSV文件"
3. **选择字段**: 勾选需要的数据列
4. **添加字段**: 可选择添加自定义字段
5. **预览数据**: 查看右侧预览区域
6. **生成文件**: 点击"生成Excel文件"

## 🔧 故障排除

### 程序无法启动
- 确保系统是64位Windows
- 尝试以管理员身份运行
- 检查是否有杀毒软件阻止

### 安全警告
- Windows可能显示"未知发布者"警告
- 点击"更多信息" → "仍要运行"

### 文件导入失败
- 确保文件格式正确（.xlsx, .xls, .csv）
- 检查文件是否被其他程序占用
- 确保文件不为空

## 📞 获取帮助
