## 📝 使用说明

### 基本工作流程
1. **启动应用程序**
   ```bash
   ./dist/ExcelDataProcessor
   ```

2. **导入数据文件**
   - 点击"选择Excel/CSV文件"按钮
   - 选择要处理的数据文件

3. **选择字段**
   - 在字段列表中勾选需要的列
   - 使用"全选"/"全不选"快速操作

4. **添加自定义字段**（可选）
   - 点击"添加新字段"按钮
   - 输入字段名称和默认值

5. **预览数据**
   - 在右侧预览区域查看处理结果
   - 确认数据格式正确

6. **生成输出文件**
   - 点击"生成Excel文件"按钮
   - 选择保存位置和文件名

## 🛠️ 故障排除

### 常见问题

1. **应用程序无法启动**
   - 检查文件权限：`ls -la ExcelDataProcessor`
   - 设置执行权限：`chmod +x ExcelDataProcessor`
   - 检查系统兼容性

2. **文件导入失败**
   - 确保文件格式正确 (.xlsx, .xls, .csv)
   - 检查文件是否损坏
   - 确保文件不是空的

3. **内存不足**
   - 处理大文件时可能需要更多内存
   - 尝试处理较小的数据集
   - 关闭其他应用程序释放内存

4. **权限问题**
   - 确保对输入文件有读取权限
   - 确保对输出目录有写入权限
